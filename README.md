# Frontend assignment info

## 프로그래밍 언어

- JavaScript

## 라이브러리 버전

| name              | version |
| ----------------- | ------- |
| react             | 18.2.0  |

| styled-components | 5.3.5   |

## 사용법

- 본 레포지토리가 있는 경로로 갑니다.
- 터미널에 `npm install`을 입력해 필요한 라이브러리를 설치합니다.
- 이후 `npm start`를 통해 브라우저 상에서 구현된 섹션을 바로 확인할 수 있습니다.
- 동작을 재확인 하고 싶으시면 새로고침을 사용하시면 됩니다.
